== Description ==

This plugin enables you to easily include your TradeMe listings or feedback directly on your site. You can also include
the result of any custom search very easily.

== Installation ==

= Automatic installation =
This plugin can be easily installed from your WordPress dashboard. Go to the Plugins section and click Add New, find
the plugin and add it.

= Manual installation =

1. Upload `trademe-widget.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
